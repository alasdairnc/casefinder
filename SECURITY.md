# Security Policy

## Supported Versions

CaseFinder is a portfolio project. Only the latest version deployed at casedive.ca is maintained.

## Reporting a Vulnerability

If you find a security vulnerability, please do not open a public GitHub issue.

Email: contact@casedive.ca

Include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact

I'll respond within 7 days. If confirmed, I'll work on a fix and credit you if you'd like.
