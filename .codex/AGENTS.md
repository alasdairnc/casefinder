# CaseFinder Codex Agents

Specialized agent roles for CaseFinder development and analysis.

## Agents

### Explorer
Fast codebase search and Q&A agent. Explores structure, finds relevant code, answers questions about architecture.

### Reviewer  
Code review and quality assurance agent. Checks style, security, performance, and adherence to project rules.

### Docs-Researcher
Documentation and legal knowledge expert. Researches CanLII case law, Criminal Code sections, Charter rights, and civil law.
